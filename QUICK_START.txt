📦 INSTALLAZIONE E AVVIO LOCALE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  Entra nella cartella del progetto:
    cd gym-tracker-app

2️⃣  Installa le dipendenze:
    npm install

3️⃣  Avvia l'app:
    npm run dev

4️⃣  Apri il browser su:
    http://localhost:3000

✅ L'app è pronta all'uso!

🚀 PASSO 2: Deploy su Vercel (FACILISSIMO!)

Opzione A - Interfaccia Web (CONSIGLIATO):
  1. Vai su vercel.com
  2. Fai login con GitHub
  3. Clicca "New Project"
  4. Seleziona "gym-tracker-app"
  5. Clicca "Deploy"
  6. ✨ FATTO! Link pronto in 1 minuto

Opzione B - Da terminale:
  npm install -g vercel
  vercel login
  vercel --prod

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 ALTERNATIVA: Deploy su Netlify

  1. npm run build
  2. Vai su netlify.com
  3. Trascina la cartella "dist" nel sito
  4. ✨ ONLINE in 10 secondi!

🆘 PROBLEMI?

• Errore "npm not found"?
  → Installa Node.js da nodejs.org

• Errore durante npm install?
  → Prova: rm -rf node_modules && npm install

• L'app non si carica?
  → Controlla che la porta 3000 sia libera
